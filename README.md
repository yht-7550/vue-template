# vue-template

基于Vue3 + Vite6 + Pinia 开发的基础项目模板

## 安装使用步骤 📔

### 安装依赖
```
pnpm install
```

### 本地运行

```
pnpm dev
```

### 项目打包

```
pnpm build
```

### 代码格式化

```
pnpm lint
```
